# Dog Care

Мобильное приложение на Expo Router для ведения журналов прогулок, кормлений и визитов к ветеринару. На странице каждой активности есть карточка-герой, блок статистики, форма ввода и swipe‑список записей с жестами удаления.

## Основные возможности

- **Прогулки** — запись длительности, заметок и автоматическое форматирование карточек по длительности.
- **Кормления** — учет веса порций, типа корма и подсчет суммарных/средних значений.
- **Ветеринар** — журнал посещений с клиниками и заметками.
- **Общие UI-компоненты** — HeroCard, StatsBlocks, TimeRecorder, SwipeableCardsList и кастомные инпуты для единообразного дизайна.

## Требования

- Node.js 18+
- npm 9+ (или pnpm/yarn)
- Expo CLI (`npx expo ...`)

## Установка и запуск

```bash
npm install
npx expo start
```

Далее выберите нужную платформу (iOS/Android/Web) в меню Expo.

## Структура проекта

```
app/                 # Маршруты Expo Router (табы walks/feeding/vet)
src/components/      # Переиспользуемые UI-компоненты
src/hooks/           # Хуки для карточек (градиенты, подписи и т.п.)
src/storage/         # JSON-хранилище в AsyncStorage
src/utils/           # Утилиты (генератор uid и т.д.)
```

## Скрипты npm

| Команда            | Назначение                    |
| ------------------ | ----------------------------- |
| `npm start`        | Expo Dev Tools                |
| `npm run ios`      | Запуск в iOS симуляторе       |
| `npm run android`  | Запуск в Android эмуляторе    |
| `npm run web`      | Запуск в браузере             |
